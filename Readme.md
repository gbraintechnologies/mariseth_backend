# Mariseth Farms

## Features
- **User Accounts Management**: Create and manage user accounts with role-based access control.

## Getting Started

### Prerequisites
- Docker installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone git@bitbucket.org:LogicielEngineer/mariseth-backend.git
   cd mariseth-backend
   ```

2. Create the main branch:
   ```bash
   git checkout development
   ```
3. Run the application:
   ```bash
   docker-compose -f docker-compose.local.yml up --build
   ```

4. Set up the organization:
   - Create the user.
   - Create the custom types.
